Limitações gerais

• RAIS: a partir de 2022 a metodologia de coleta mudou (base eSocial); os perfis detalhados (escolaridade, ocupação, tempo de emprego, massa salarial) têm uma quebra de série entre 2021 e 2022, por isso são tratados como retrato do último ano quando não há série completa.

• CAGED: cobre a janela indicada dinamicamente acima (calculada a partir dos dados carregados), com granularidade mensal por UF/CNAE.

• Comtrade (comércio mundial): classificação por código HS como proxy do CNAE, já que o Comtrade não reporta por CNAE. Usado apenas para contexto (Brasil x Mundo), não como número oficial de comércio exterior do Brasil; para isso, ver Comex (MDIC).

• Dados financeiros (PIA/IBGE): a base "Fundição 24.5" reúne 2451 e 2452 sem abertura entre eles; é usada nos dois segmentos com a mesma ressalva.

• Energia (CCEE): consumo exato por CNAE só está disponível a partir de abr/2024; a série "Metalurgia e Produtos de Metal" é uma categoria mais ampla usada como contexto adicional, e é a única fonte disponível para a quebra Livre x Autoprodutor.

• DECOM: base específica de ferro e aço (2451); não há processos catalogados para não ferrosos (2452) nesta fonte.

• Todos os valores monetários estão em valores correntes (nominais), exceto onde explicitamente indicado como deflacionado (remuneração real, bloco Emprego formal).

Dados consolidados de fontes públicas (IBGE, RAIS/CAGED, MDIC/Comex Stat, UN Comtrade, CCEE, BNDES, DECOM/GECEX). Painel elaborado em parceria com FIESP e SENAI.
